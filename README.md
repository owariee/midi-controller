# midi-controller

This is a midi controller firmware implementation for Arduino Mega.

## Requirements

### Arch Linux

```sh
$ pacman -S arduino-cli
```
or
```sh
$ yay -S arduino-cli
```

## Install dependencies, build and upload code

In your terminal:

```sh
$ cd mega2560
$ make deps
$ make build
$ make upload
```

or

```sh
$ cd mega2560
$ make
```
